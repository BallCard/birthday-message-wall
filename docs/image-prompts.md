## Usage Notes

1. **Remove `--ar` flags** if your generator doesn't support aspect ratio
2. **Add `--no text`** if your generator tends to add unwanted text
3. For sprite sheets: generate at high res, then slice into individual frames using an image editor
4. All assets should be exported as **PNG with transparent background** for web use
5. Recommended final sizes:
   - Avatars: 64x64 or 128x128
   - Gift icons: 32x32 or 48x48
   - Pet sprites: 60x60 per frame
   - Background: match viewport, compress as JPG
   - Card texture: 256x256, repeat/tile
